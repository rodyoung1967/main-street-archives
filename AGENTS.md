# Agent Instructions

These instructions apply to any AI or automation working on The Main Street Archive.

## Core Rules

- Never delete historical information.
- Never renumber existing IDs.
- Always update related files when new information is added.
- Preserve uncertainty rather than inventing certainty.
- Prefer traceability over brevity.

## Adding People

If a new person is introduced:

- Add the person to `database/people.yml`.
- Update `people/people-index.md`.
- Connect the person to related businesses and buildings.

## Adding Artifacts

If a new artifact is added:

- Assign an `A-###` ID.
- Link the artifact to supporting evidence.
- Link the artifact to related buildings and businesses.

## Updating Evidence

If evidence changes:

- Update the Evidence Register.
- Update the Timeline if appropriate.
- Leave conflicting evidence intact until resolved.
