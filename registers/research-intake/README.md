# Research Intake

Drop ChatGPT **Research Session Handoff** files here before Cursor integration.

## Naming

`YYYY-MM-DD-short-topic.md`

Example: `2026-07-04-sanborn-1925-501-503-505.md`

## Workflow

1. Research in ChatGPT project **Main Street**.
2. Say **handoff** at end of session.
3. Save the markdown file here.
4. In Cursor: *Integrate this into the archive per WORKFLOW.md*
5. Run `python scripts/validate_archive.py`

Handoff template and rules: [`docs/CHATGPT_PROJECT.md`](../../docs/CHATGPT_PROJECT.md)
