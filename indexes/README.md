# Indexes

This folder contains cross-register navigation files for The Main Street Archive.

## Files

- `id-crosswalk.md` — connects archive IDs across buildings, businesses, people, evidence, sources, media, artifacts, and Sanborn maps.

## Maintenance

Update `id-crosswalk.md` whenever a source, evidence item, media item, artifact, person, business, building, or map changes in a way that affects ID relationships.

Canonical detailed registers remain in their subject folders:

```text
evidence/evidence-register.md
evidence/source-register.md
media/photo-metadata-register.md
maps/sanborn-map-register.md
people/people-index.md
businesses/business-index.md
```
