## Image Handling Notes

- Preserve the original source URL in `evidence/source-register.md`.
- If an image is downloaded, keep the filename tied to the artifact ID.
- Do not crop, edit, or enhance the archival copy.
- If an edited derivative is useful, save it as a separate derivative file and document the change.
- Record image owner, collector, page URL, download date, and any visible marks or captions when known.
